# Mechathon

This repository contains source code of Sprinteři27's robot for [Mechathon
2020](https://www.mechathon.cz/).

## Installation

```cmd
py -3 -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -r requirements.txt
```
